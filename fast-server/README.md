# Fast Server

## How to run
1. `.env` 파일 생성
   ```
   HOST=
   PORT=
   ELASTIC_USERNAME=
   ELASTIC_PASSWORD=
   MOVIE_INDEX='movie'
   DRAMA_INDEX='drama'
   PEOPLE_INDEX='people'
   GENRE_INDEX='genre'
   KEYWORD_INDEX='keyword'
   COMPANY_INDEX='company'
   RATING_INDEX='rating'
   FIELD_INDEX='field'
   CHANNEL_INDEX='channel'
   ```
2. FastAPI 실행
   Dockerfile 참조
